Tentang Saya

Saya Nabil Ahmad Firdaus, Saya adalah siswa kelas 11 di SMKN 1 Surabaya, jurusan Rekayasa Perangkat Lunak (RPL). Di jurusan ini, saya mempelajari berbagai keterampilan dalam pemrograman dan pengembangan perangkat lunak, yang mendukung minat saya dalam teknologi. Saya memiliki hobi bermain game dan menonton film, saya juga suka mendengarkan musik dan membaca buku.
